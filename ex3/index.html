This page demonstrates a basic layout using CSS Grid with grid-template-areas.

What is CSS Grid?

CSS Grid is a powerful layout system that allows you to create two-dimensional layouts with rows and columns.

Grid Template Areas

This page uses grid-template-areas to define named grid areas. Check the CSS to see how each section is placed:

  • navbar: Takes the full width at the top
  • sidebar: Left column for navigation
  • main: Main content area
  • footer: Takes the full width at the bottom

Practice Exercise

Try inspecting this page's CSS to see how the grid layout is created. Pay attention to:

  • grid-template-areas
  • grid-template-columns
  • grid-template-rows
  • gap property
